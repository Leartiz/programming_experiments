# Priority queue & heap

## Priority queue vs heap

**Priority queue** - абстракция: вставить элемент, забрать "главный" (min или max по приоритету).

**Heap (куча)** - частая реализация: бинарное дерево в слайсе, операции за O(log n).

В Go готовой `PriorityQueue` нет - есть пакет [`container/heap`](https://pkg.go.dev/container/heap): алгоритмы кучи поверх **твоего** слайса.

## Min-heap в массиве

Свойство: **родитель <= оба ребёнка**.
Минимум всегда в `h[0]`.

Для индекса `i`:

```
родитель:       (i - 1) / 2
левый ребёнок:  2*i + 1
правый ребёнок: 2*i + 2
```

Пример `[1, 3, 2, 5, 4]`:

```
        1
      /   \
     3     2
    / \
   5   4

индексы: 0=1  1=3  2=2  3=5  4=4
```

**Max-heap** - то же, но `Less` перевёрнут: `h[i] > h[j]`.

## Операции (пакет container/heap)

| Функция | Что делает |
|---------|------------|
| `heap.Init(h)` | превратить слайс в кучу, O(n) |
| `heap.Push(h, x)` | append + просеивание вверх |
| `heap.Pop(h)` | снять минимум (`h[0]`), просеивание вниз |
| `heap.Remove(h, i)` | удалить элемент с индексом `i`, вернуть его; O(log n) |
| `heap.Fix(h, i)` | восстановить кучу после изменения `h[i]` |

## heap.Interface

Тип должен реализовать:

```go
type Interface interface {
	  sort.Interface
	  Push(x any) // add x as element Len()
	  Pop() any   // remove and return element Len() - 1.
}

//...

type Interface interface {
	Len() int
	Less(i, j int) bool
	Swap(i, j int)
}
```

| Метод | Роль |
|-------|------|
| `Len()` | размер слайса |
| `Less(i, j)` | **приоритет**: для min-heap `h[i] < h[j]` |
| `Swap(i, j)` | перестановка при просеивании |
| `Push(x)` | `append` в конец |
| `Pop()` | убрать **последний** элемент, вернуть его |

### Важно: два разных Pop

- **`heap.Pop(h)`** (пакет) - "достань минимум из priority queue", это то, что вызываешь в коде.
- **`(*IntHeap).Pop()`** (твой метод) - внутренний: отрезать хвост слайса. Пакет сам кладёт бывший последний элемент в корень и просеивает.

### Поток heap.Pop (упрощённо)

```
1. answer = h[0]
2. h[0] = последний элемент
3. твой Pop() - укоротить слайс
4. sift down от корня (Less + Swap)
```

## Связь с C++

| Go | C++ |
|----|-----|
| `container/heap` + свой тип | `std::priority_queue` из коробки |
| приоритет в `Less` | компаратор / `std::greater` для min-heap |

## Когда применять

- k наименьших/наибольших элементов
- Dijkstra, merge k sorted lists
- задачи "всегда обрабатывай следующий по приоритету"
